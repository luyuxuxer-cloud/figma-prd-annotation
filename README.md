# Figma PRD Annotation

A Codex skill for creating concise, PRD-based Figma annotations for product review, design review, and engineering handoff.

Use it to:

- Read a final PRD and map key rules to Figma frames.
- Add lightweight annotations for rules, states, exceptions, and open questions.
- Keep the canvas readable instead of turning it into another PRD.
- Prepare design files for review and handoff.

## Installation

Copy this folder into your Codex skills directory:

```bash
cp -R figma-prd-annotation ~/.codex/skills/
```

## Usage

```text
Use $figma-prd-annotation to annotate this Figma design from the final PRD.
```
